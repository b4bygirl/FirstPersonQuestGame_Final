# 3D Игра от первого лица: Everland

![Gameplay](Assets/GIFs/gameplay.gif)

3D квест, сделанный в Unity.

## Описание
Проект демонстрирует игру от первого лица с множеством интерактивных механик.  
Цель — показать навыки программирования, работы со сценами и создания игрового процесса.

## Особенности
- Передвижение от первого лица
- Возможность рисовать
- Движение на лодке
- Взаимодействие с NPC
- Бои с врагами
- Система заданий (отмечать выполнение задач внутри зданий)
- Переход между локациями (комната ↔️ лес)
- Звуки шагов
- Карта для навигации по местности
- Сбор и решение пазлов
- Взаимодействие с компьютером (можно сесть и использовать)
- Другие интерактивные объекты окружения

## Структура проекта
- `Assets/Scripts/` – все скрипты на C#  
- `Assets/Scenes/` – игровые сцены  
- `ProjectSettings/` – настройки проекта Unity  
- `Packages/` – Unity пакеты

## Технологии
- Unity  
- C#

## Примечания
- Тяжёлые ассеты (модели, текстуры, аудио) **не включены**, чтобы репозиторий был лёгким и сосредоточенным на коде.  
- Этот проект демонстрирует навыки программирования, проектирования игрового процесса и организации проекта.
