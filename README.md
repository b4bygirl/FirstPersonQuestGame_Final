# 3D игра от первого лица: Everland

![Gameplay](Assets/GIFs/gameplay.gif)

**Everland** — 3D-квест от первого лица, разработанный в Unity.  
Проект создан как демонстрация навыков программирования, работы с игровыми механиками.

## Описание
Проект демонстрирует игру от первого лица с множеством интерактивных механик и переходом между локациями.  
Основная цель проекта — показать умение разрабатывать игровой процесс, системы взаимодействия и логику поведения объектов.

## Особенности
- Передвижение от первого лица
- Возможность рисовать в игровом мире
- Движение на лодке
- Взаимодействие с NPC
- Боевая система с врагами 
- Система заданий (отслеживание выполненных задач)
- Переход между локациями (комната ↔️ лес)
- Звуки шаговв зависимости от поверхности 
- Карта для навигации по игровому миру 
- Сбор и решение пазлов
- Взаимодействие с компьютером (можно сесть и использовать)
- Другие интерактивные объекты окружения

## Структура проекта
- `Assets/Scripts/` – все скрипты на C#  
- `Assets/Scenes/` – игровые сцены  
- `ProjectSettings/` – настройки проекта Unity  
- `Packages/` –  используемые Unity пакеты

## Технологии
- Unity  
- C#

## Примечания
- Тяжёлые ассеты (модели, текстуры, аудио) **не включены** в репозиторий, чтобы сосредоточиться на коде и логике проекта.
- Проект демонстрирует навыки:
  - программирования на C#  
  - проектирования игровых механик  
  - работы со сценами и состояниями игры  
  - организации структуры Unity-проекта
